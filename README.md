# buttweet
<GreedButt.com> からのデータを取得してtwitterに投稿する。


# Install
```sh
% ./main.py --twitter-auth
```

# Usage
## 毎日の結果を投稿する
```sh
% ./main.py
```

## 数日分の結果をまとめて投稿する
(未実装)
* 一週間分の結果を投稿する
```sh
% ./main.py --summary-mode 7
```
